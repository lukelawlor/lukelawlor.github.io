asteroids-js

asteroids-js is an asteroids clone game made in JavaScript with the HTML5 canvas. The goal is to last as long as possible while destroying asteroids that increasingly fill the screen.

The game also comes with an in-depth tutorial that teaches people how the game works and how to make their own JavaScript games.

This game was made for my officer application for my school's coding club over the period of ~4 days.
