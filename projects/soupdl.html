SoupDL 06

SoupDL 06 (stupid name I know) is a platformer and shooter game and engine I've been working on. It's written in C and uses SDL2's renderer for graphics.

Currently it features:

  • An ECS-like entity system
  • A tilemap system
  • A map editor

This has been the most fun and ambitious project I've worked on so far.
